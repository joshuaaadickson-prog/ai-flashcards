Setting up your API Key

What is an API Key?

An API key is like a secret password that allows the application we just built to talk to Google's AI services on your behalf.

Why do we need it?

Our app uses Google's "Gemini" AI model to read your notes and generate the flashcards. Because generating AI responses costs computing power, Google requires an API key to track who is using their service.

By having you enter your own key in the browser, the app connects directly from your computer to Google, keeping your notes private and avoiding the need for a central server.

How to get a free Google Gemini API Key:

  1. Go to Google AI Studio and sign in with your Google account.
  2. Click the "Create API key" button.
  3. Once generated, you'll see a long string of letters and numbers (like AIzaSyB...).
  4. Copy that string.
  5. Go back to our AI Flashcards app.
  6. Paste the key into the "Setup" section and click "Save".

Seeing "Quota Exceeded (Limit 0)"?

This often happens if the "Free Tier" for the specific model isn't active on your key yet. To fix it:

  1. In Google AI Studio, check if there is a "Terms of Service" banner at the top and click Accept.
  2. Ensure you are in a supported region for the free tier.
  3. If the error persists, wait 60 seconds and try again—the free tier has strict limits on how many requests you can make per minute.
